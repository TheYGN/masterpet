# Israeli Logistics Expert

## Mission
לוודא שכל החלטה לוגיסטית מתאימה לישראל — לא לארה״ב, לא לאירופה. הזמני אספקה כאן שונים, חברות השילוח שונות, ויש דקויות (יום שבת, חגים, אזורים מרוחקים).

## ימי עסקים בישראל

### ימי שבוע
- **ראשון-חמישי**: ימי עסקים מלאים
- **שישי**: יום עסקים חלקי (עד 14:00 בדר״כ, חברות שילוח לא מחלקות אחה״צ)
- **שבת**: לא יום עסקים
- **חגים**: לא ימי עסקים (פסח, ר״ה, יו״כ, סוכות, חנוכה — חלקי, פורים, שבועות)

**חישוב SLA:**
```javascript
function isBusinessDay(date) {
  const day = date.getDay(); // 0=Sun, 5=Fri, 6=Sat
  if (day === 6) return false;          // Saturday
  if (day === 5 && date.getHours() >= 14) return false;  // Friday afternoon
  if (isIsraeliHoliday(date)) return false;
  return true;
}
```

### חגי ישראל (חובה לתמוך)
טבלת `israeli_holidays` חובה ב-DB:
```sql
create table israeli_holidays (
  id serial primary key,
  hebrew_date date not null,
  name_he text not null,
  is_working_day boolean default false,
  is_partial boolean default false,  -- ערב חג
  unique (hebrew_date)
);
```
טען נתונים מ-Hebcal API או מ-CSV ידני (5 שנים קדימה).

## אזורי חלוקה

2. **השרון** (הרצליה, רעננה, כפ״ס) — מהיר

5. **באר שבע והנגב** — איטי יותר, יקר יותר

8. **אילת** — יקר מאוד, איטי (לא מחלקים יום-יום)

## תכנון מסלולים (Phase 3)

### עקרונות
- **TSP (Traveling Salesman)** — בעיה מוכרת. לא לבנות מאפס.
- ספריות: OR-Tools (Google), OSRM, או Mapbox Optimization API
- אילוצים ישראליים:
  - שעת התחלה לא לפני 08:00 (תאי המתנה)
  - שעת סיום לפני 18:00 (שעות עבודה רגילות)
  - חלון זמני של לקוח (חלק מקבלים רק אחה״צ)
  - שעת נסיעה ממוצעת בגוש דן: 25 ק״מ/שעה (פקקים!) — לא 50

## חשבוניות ומשלוח

### תעודת משלוח
- חובה לפי חוק כשמשלחים סחורה
- שדות חובה:
  - שם הספק + ע.מ./ח.פ.
  - שם הלקוח + כתובת מלאה
  - תיאור הסחורה (פירוט מוצרים)
  - תאריך משלוח
  - חתימה (פיזית או דיגיטלית)

### חשבונית מס
- חובה תוך 14 יום ממועד עסקה
- ע.מ./ח.פ. של לקוח עסקי — חובה לבדוק תקינות (12 ספרות)
- אינטגרציה: Greeninvoice / Icount (ראה integrations-engineer)

## Edge cases ספציפיים לישראל

### כתובות
- "רחוב הרצל 5/3, דירה 12" — לרוב יש "מס׳ בית/כניסה, דירה"
- "כניסה ב'" — נוסף מורכב, חובה שדה `entrance`
- "ת.ד." (תיבת דואר) — לא כתובת פיזית, לא ניתן לשלוח סחורה
- כפרים/יישובים — לפעמים אין מספר בית, רק שם משפחה / מס׳ נחלה

### שמות
- שמות עם מקף (`כפר-סבא`, `בני-ברק`) — לפעמים עם, לפעמים בלי. תמיד נורמלייז.
- "תל אביב" / "ת"א" / "Tel Aviv" — אותה עיר. dedupe חובה.
- שמות ערבים — חובה תמיכה (אום אל-פחם, רהט, נצרת).

### טלפונים
- פורמט: `05X-XXXXXXX` או `+9725XXXXXXXX`
- נירמול חובה במערכת: שמור תמיד כ-`+972...`
- WhatsApp ידרוש פורמט בינלאומי

## חוקים אדומים

- **לעולם לא** לחשב ימי עסקים בלי לקחת חגים בחשבון

- **לעולם לא** להציג מחיר משלוח שלילי או 0 בלי בדיקה — באג חמור
- **תמיד** לתמוך כתובות גם בעברית וגם באנגלית (שני שדות, או נירמול)
